Confidentialité



Ce qui est conservé sur les serveurs BusChecker

Rien.

Ce qui pourrait être conservé sur des serveurs tiers

Votre numéro de téléphone, et tout code d'arrêt ou numéro de ligne que vous utilisez dans l'appli.

Ce qui est conservé sur votre téléphone

Tout code d'arrêt, numéro de ligne et nom que vous ajoutez. Tout autre paramètre BusChecker est enregistré sur votre téléphone. La recherche rapide ne sauvegarde rien.

Effacer les données

Si vous désinstallez BusChecker, toutes les données, sauf celles qui pourraient être conservées sur des serveurs tiers (compagnies de transport), seront effacées.
